在Koding云平台上搭建Hexo环境

=

  1. -

利用 Koding 云平台，用户无需在本地安装配置Node.js 环境。 虽然需要配置 Koding 上的环境，但由于环境配置在云端，可以跨平台编写发布，只要有网络，手机，平板都可以操作！

Koding 介绍

Koding 是个云端开发与社区平台，提供 Linux 虚拟主机环境及完整的 WebUI， 可以通过浏览器在 Koding 上进行主机配置、假设网站与系统开发测试等。

  • koding 提供完整 Linux 操作系统 (Ubuntu 13.04, Raring Ringtail)
  • 提供 sudo 以安装系统软件与操作
  • 预装 npm
  • 预装 git
  • 预装 apache

koding.com这是一个免费的云端开发平台，也相当于通过浏览器去操作一个VM (virtual machine)，可以完全依照自己的需要去部属开发环境、架设网站等等，这主机也可以执行JavaScript、PHP、Perl、Python、Ruby等语言，写完可以在线上预览、执行。不过要注意的是有基本容量限制3G。

他也有社交的功能，就像一般社交网站一般，可以发文分享、讨论交流，也有为数不少不同类型的Group，他把社交化功能与Web IDE结合得还不错。遇到一些开发相关问题，也还可以通过线上的社群直接发问。
